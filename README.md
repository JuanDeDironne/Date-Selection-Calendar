# Date Selection by Calendar

### Description
JavaScript and CSS code to display a simple and user-friendly calendar.
With simple configuration in the JavaScript code (ds_cal.js) if needed for a few adjustments. And a dedicated CSS (ds_cal.css) for easy color changes

### How to use...?
Link <b>ds_cal.css</b> and add script <b>ds_cal.js</b> in your page.
<br />
Add the class <b>ds-cal-input-date</b> to the input tags where you want to use the calendar selector.<br />
Example :<br />
```<input type="text" class="ds-cal-input-date" id="birthdate">```

And insert (just once) the line <b>ds_cal_use()</b> at the bottom of your page (in a script tags).<br />
Example :<br />
`<script>ds_cal_use()</script>`
<br /><br />
See the full example in index.html.
